About

Who we are

Akhenaten is a community-driven effort to make the 1999 game Pharaoh compatible with modern systems. It features a redesigned engine with support for loading original save games and playing campaign missions.

The project is a fork of Augustus, aimed at supporting Pharaoh, with a new renderer, scripts, and game mechanics. It's an ongoing project that welcomes contributors.

History

Pharaoh (1999) was developed by Impressions Games. Development began in fall 1997, using the Caesar 3 engine as a foundation. The game sold 1.7 million copies at $45 each on a sub-$2M budget.

Rights passed through multiple owners; Dotemu later released Pharaoh: A New Era remake. Dalerank took over the open-source recreation after the original author abandoned it.

Technical

  • New deferred renderer with Y-sorting
  • C++ core with JavaScript configs for buildings and figures
  • Mod support — download mods from GitHub, load textures from modpacks
  • Builds for Windows, macOS (ARM/x86), Linux, Android, WebAssembly
  • Original .sg3/.555 texture format support
