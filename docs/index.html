Akhenaten

Open Source Pharaoh + Cleopatra Engine

Build and manage an ancient Egyptian civilization along the Nile. Resource management, city planning, infrastructure, and the welfare of your citizens.

Download GitHub Play in Browser

You must own a copy of the original game to use this project.

Original Mechanics Restored

Full reimplementation of Pharaoh + Cleopatra gameplay — original save games, campaign missions, and all core systems.

Modern Platforms

Runs on Windows, macOS, Linux, Android, and in the browser via WebAssembly. Nightly builds available.

Open Source

Non-commercial, community-driven. Fork of Augustus adapted for Pharaoh with a new renderer and engine.

Active Development

New buildings, enemies, language support, mod system, and developer tools added throughout 2025.
